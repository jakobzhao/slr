Introduction

Navigate map with cursor or use the "Zoom to State" feature in the navigation bar. Hover mouse over hexagon to see migration from the inundated region. Magnitude of migration groups are represented by arc thickness (???).


Migration data based on estimates for 1.8m of sea level rise, and for estimated U.S. population in 2100. See "Team and Project Information" for more details.
